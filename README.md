# 链家网新房数据可视化及预测系统 python1024

## 项目概述

本项目是基于 Python 与 Flask 框架，运用机器学习算法实现的对链家网新房数据的可视化及预测系统。系统爬取链家全国新房数据，为用户提供数据分析和预测功能。

## 技术栈

- **爬虫**: 使用 `requests` 库进行数据爬取
- **数据库**: 采用 `MySQL` 进行数据存储
- **后端**: 基于 `Flask` 框架进行开发
- **机器学习算法**: 使用 `scikit-learn` 库实现多元线性回归进行预测
- **数据可视化**: 使用 `Echarts` 进行数据可视化展示

## 功能模块

- **数据爬取**: 爬取链家新房的18个字段信息
  - 房名、封面、市区、地区、详细地址
  - 装修情况、公司、房屋类型、交房时间、开盘时间
  - 标签、总价区间、售房情况、详情链接

- **数据存储**: 将爬取的数据存储到 MySQL 数据库
- **数据预测**: 利用机器学习算法对新房数据进行预测分析
- **数据可视化**: 使用 Echarts 将分析结果以图表形式直观展示

## 系统角色

- **数据爬取端**: 负责从链家网获取新房数据
- **服务端**: 负责数据处理、存储、预测分析及前端数据供给
- **客户端**: 展示数据可视化结果，提供用户交互界面

## 运行环境

- **操作系统**: 兼容 Linux 与 Windows
- **编程语言**: Python 3.*
- **依赖管理**: pip, virtualenv 或 conda
- **数据库**: MySQL

## 部署步骤

1. 环境准备：安装 Python, pip, virtualenv, MySQL
2. 克隆项目代码到本地
3. 初始化数据库，并创建相应的表结构
4. 使用 `requirements.txt` 安装项目依赖
5. 配置项目相关环境变量及数据库连接
6. 运行爬虫程序
7. 启动 Flask 服务
